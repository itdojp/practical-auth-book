## 🎨 プレビューコンテンツ

### 1. 📐 レイアウト構造
```
┌─────────────────────────────────────────┐
│ Header (検索・テーマ切り替え・GitHubリンク)    │
├─────────────┬───────────────────────────┤
│ Sidebar     │ Main Content              │
│ - 目次      │ - パンくずリスト           │
│ - 進捗表示   │ - 記事コンテンツ           │
│ - 章リスト   │ - コードブロック           │
│             │ - ページナビゲーション       │
└─────────────┴───────────────────────────┘
```

**特徴:**
- ✅ 固定ヘッダーで常にナビゲーション可能
- ✅ サイドバーは折りたたみ可能（モバイル対応）
- ✅ 進捗バーで読書の進行状況を可視化
- ✅ レスポンシブデザインで全デバイス対応

### 2. 📝 タイポグラフィシステム

```css
/* フォントサイズ階層 */
h1: 36px (2.25rem) - メインタイトル
h2: 30px (1.875rem) - セクションタイトル  
h3: 24px (1.5rem) - サブセクション
本文: 16px (1rem) - 読みやすさ重視
小文字: 14px (0.875rem) - キャプション・UI
```

**特徴:**
- ✅ 75文字幅で最適な可読性
- ✅ 1.5の行間で読みやすさ向上
- ✅ システムフォント使用で高速表示

### 3. 💻 コードブロック

```javascript
// 実際のプレビューでは以下の機能が動作します
const codeBlock = {
  syntaxHighlighting: true,
  copyButton: true,
  languageLabel: "JavaScript",
  theme: "GitHub風"
};
```

**機能:**
- 🎨 **シンタックスハイライト**: 言語別の色分け
- 📋 **コピーボタン**: ワンクリックでクリップボードにコピー
- 🏷️ **言語ラベル**: コードの種類を明示
- 📱 **モバイル対応**: 横スクロール、タッチ最適化

### 4. 🧭 ナビゲーションシステム

#### パンくずリスト
```
🏠 ホーム › 章 › 第1章: 基本概念
```

#### ページナビゲーション
```
┌─────────────┬─────────────┬─────────────┐
│ ← 前へ      │   📚 目次    │    次へ →   │
│ はじめに     │             │  第2章: 実践編│
└─────────────┴─────────────┴─────────────┘
```

**特徴:**
- ✅ 直感的な前後移動
- ✅ 現在位置の明確化
- ✅ ホバーエフェクトで視覚的フィードバック

## 🔄 インタラクティブ機能

### テーマ切り替え
1. 右上の 🌙/☀️ ボタンをクリック
2. ライトモード ⇄ ダークモード の切り替え
3. システム設定の自動検出にも対応

### サイドバー操作
1. 左上の ☰ ボタンでサイドバー切り替え（モバイル）
2. 章をクリックして直接ジャンプ
3. 進捗バーで現在の読書位置を確認

### コードブロック
1. コードにホバーすると「Copy」ボタンが表示
2. クリックでクリップボードにコピー
3. 成功時に「Copied!」表示
