# 第1章: 基本機能

この章では、v2.0テンプレートの基本機能について説明します。

## 1.1 Easy Setup

easy-setup.jsスクリプトにより、対話式でプロジェクトの初期設定が可能です：

```bash
node easy-setup.js
```

## 1.2 簡単ビルド

build-simple.jsにより軽量なビルドが実行できます：

```bash
node scripts/build-simple.js
```

## 1.3 コードハイライト

```javascript
// JavaScript example
console.log("Hello, World!");

function buildBook() {
    return "Built successfully!";
}
```

```python
# Python example
def hello_world():
    print("Hello, World!")
```

## 1.4 数式サポート
